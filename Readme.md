# Звичайні жорданові виключення (ЗЖВ)

## Опис

Цей проєкт реалізує метод **звичайних жорданових виключень** для розв’язання задач лінійної алгебри:

- знаходження оберненої матриці
- обчислення рангу матриці
- розв’язання систем лінійних алгебраїчних рівнянь (СЛАР)

Також реалізовано покроковий протокол обчислень.

---

## Функціонал

- обернена матриця
- ранг матриці
- розв’язання СЛАР через обернену матрицю
- протокол виконання кроків

---

## Запуск

1. Відкрити `index.html` у браузері
2. (За потреби) скомпілювати TypeScript:

```bash
tsc app.ts
```
